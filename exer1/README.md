# Exercise 1

## Note
Please do not use **AI** to implement this assignment.
Ensure you create another branch called `clas1` for this exercise
Ensure the files are in this directory and file names are at the end of each instructions

1. Write a program that prints your name 100 times. (names.py)
2. Write a program to fill the screen horizontally and vertically with your name. [Hint: add the
option end='' into the print function to fill the screen horizontally.] (screen.py)
3. Write a program that outputs 100 lines, numbered 1 to 100, each with your name on it. The
output should look like the output below. (number_name.py)
```
1 Your name
2 Your name
3 Your name
4 Your name
...
100 Your name
```
4. Write a program that prints out a list of the integers from 1 to 20 and their squares. The output
should look like this (integers.py)
```
1 --- 1
2 --- 4
3 --- 9
...
20 --- 400
```
5. Write a program that uses a for loop to print the numbers 8, 11, 14, 17, 20, . . . , 83, 86, 89. (sequence.py)
6. Write a program that uses a for loop to print the numbers 100, 98, 96, . . . , 4, 2. (even.py)
