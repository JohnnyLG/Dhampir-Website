# Assignment 1

### Task 1

- **Given:** A list of strings: `['Hello', 'from', 'the', 'other', 'side']`
    1. Create a Python script named `echo.py`.
    2. Inside the script, define the list of strings.
    3. Concatenate the strings in the list to form the sentence: "Hello from the other side".
    4. Print the final sentence to the console.

### Task 2

- **Given:** A list of numbers: `[10, -5, 8, 3, -12, 7]`

    1. Create a Python script named `numbers.py`.
    2. Inside the script, define the list of numbers.
    3. Calculate the sum of all the numbers in the list.
    4. Print the result to the console.

### Task 3

- **Given:** A list of ingredients: `['Dragon's Breath', 'Unicorn's Horn', 'Mermaid's Tears']`

    1. Create a Python script named `recipe.py`.
    2. Inside the script, define the list of ingredients.
    3. Add "Phoenix Feather" to the end of the list.
    4. Remove "Unicorn's Horn" from the list.
    5. Print the final list of ingredients to the console.

### Task 4

- **Given:** A string: `'nohtyp'`

    1. Create a Python script named `riddle.py`.
    2. Inside the script, define the given string.
    3. Reverse the string.
    4. Print the reversed string to the console.

## Instructions for Submission

1.  Create a separate `.py` file for each task.
2.  Make sure your code is clean and readable.
3.  When you are done, you will have four files: `echo.py`, `numbers.py`, `recipe.py`, and `riddle.py`.
